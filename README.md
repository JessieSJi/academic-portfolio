# Academic Portfolio

Modern single-page academic personal website (React + TypeScript + Tailwind CSS v4).

## Preview

```bash
cd C:\Users\jishu\Projects\academic-portfolio
npm install
npm run dev
```

Open the URL shown in the terminal (typically `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```

## Customize

Edit sample content in `src/data/profile.ts`.
